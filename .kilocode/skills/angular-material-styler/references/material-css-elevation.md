# Material Design 3 Elevation Tokens — Справочник

Краткий справочник по системным CSS-переменным Angular Material для теней и рамок.

## Общая логика

Elevation токены определяют глубину элементов через тени и рамки. Используются через свойства `box-shadow` и `border`:

```css
.element {
  box-shadow: var(--mat-sys-level2);
  border: 1px solid var(--mat-sys-outline);
}
```

## Outline — Рамки и границы

Используются для создания границ вокруг элементов, разделителей, обводок.

| Переменная | Назначение |
|------------|------------|
| `--mat-sys-outline` | Основной цвет рамок и границ |
| `--mat-sys-outline-variant` | Альтернативный цвет рамок (менее заметный) |

**Пример:**
```css
.card {
  border: 1px solid var(--mat-sys-outline);
}

.divider {
  border: 1px solid var(--mat-sys-outline-variant);
}
```

## Level — Уровни теней

Используются для создания глубины и иерархии элементов. Чем выше уровень, тем заметнее тень.

### Level 0 — Без тени

Используется для элементов на том же уровне, что и фон, без возвышения.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-level0` | 0px 0px 0px 0px rgba(0, 0, 0, .2), 0px 0px 0px 0px rgba(0, 0, 0, .14), 0px 0px 0px 0px rgba(0, 0, 0, .12) | Без тени |

**Пример:**
```css
.flat-card {
  box-shadow: var(--mat-sys-level0);
}
```

### Level 1 — Минимальная тень

Используется для элементов с лёгким возвышением, например, для карточек на фоне.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-level1` | 0px 2px 1px -1px rgba(0, 0, 0, .2), 0px 1px 1px 0px rgba(0, 0, 0, .14), 0px 1px 3px 0px rgba(0, 0, 0, .12) | Минимальная тень |

**Пример:**
```css
.card {
  box-shadow: var(--mat-sys-level1);
}
```

### Level 2 — Небольшая тень

Используется для элементов с заметным возвышением, например, для карточек, панелей.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-level2` | 0px 3px 3px -2px rgba(0, 0, 0, .2), 0px 3px 4px 0px rgba(0, 0, 0, .14), 0px 1px 8px 0px rgba(0, 0, 0, .12) | Небольшая тень |

**Пример:**
```css
.panel {
  box-shadow: var(--mat-sys-level2);
}
```

### Level 3 — Средняя тень

Используется для элементов со значительным возвышением, например, для модальных окон, выпадающих меню.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-level3` | 0px 3px 5px -1px rgba(0, 0, 0, .2), 0px 6px 10px 0px rgba(0, 0, 0, .14), 0px 1px 18px 0px rgba(0, 0, 0, .12) | Средняя тень |

**Пример:**
```css
.dialog {
  box-shadow: var(--mat-sys-level3);
}

.dropdown {
  box-shadow: var(--mat-sys-level3);
}
```

### Level 4 — Большая тень

Используется для элементов с сильным возвышением, например, для FAB кнопок, всплывающих окон.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-level4` | 0px 5px 5px -3px rgba(0, 0, 0, .2), 0px 8px 10px 1px rgba(0, 0, 0, .14), 0px 3px 14px 2px rgba(0, 0, 0, .12) | Большая тень |

**Пример:**
```css
.fab-button {
  box-shadow: var(--mat-sys-level4);
}

.popover {
  box-shadow: var(--mat-sys-level4);
}
```

### Level 5 — Максимальная тень

Используется для элементов с самым сильным возвышением, например, для навигационных ящиков, полноэкранных модалок.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-level5` | 0px 7px 8px -4px rgba(0, 0, 0, .2), 0px 12px 17px 2px rgba(0, 0, 0, .14), 0px 5px 22px 4px rgba(0, 0, 0, .12) | Максимальная тень |

**Пример:**
```css
.drawer {
  box-shadow: var(--mat-sys-level5);
}

.fullscreen-modal {
  box-shadow: var(--mat-sys-level5);
}
```

## Рекомендации по использованию

1. **Level 0** — используйте для плоских элементов на том же уровне, что и фон
2. **Level 1** — используйте для карточек на фоне, небольших панелей
3. **Level 2** — используйте для карточек, панелей, вкладок
4. **Level 3** — используйте для модальных окон, выпадающих меню, тостов
5. **Level 4** — используйте для FAB кнопок, всплывающих окон, снекбаров
6. **Level 5** — используйте для навигационных ящиков, полноэкранных модалок
7. **Outline** — используйте для рамок карточек, полей ввода, разделителей
8. **Outline-variant** — используйте для менее заметных рамок, разделителей
