# Material Design 3 Shape Tokens — Справочник

Краткий справочник по системным CSS-переменным Angular Material для скругления углов.

## Общая логика

Shape токены определяют радиус скругления углов элементов. Используются через свойство `border-radius`:

## Extra Large — Очень большое скругление

Используется для крупных элементов с мягкими углами, например, для модальных окон или больших карточек.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-corner-extra-large` | 28px | Скругление всех углов |
| `--mat-sys-corner-extra-large-top` | 28px 28px 0 0 | Скругление только верхних углов |

## Extra Small — Очень маленькое скругление

Используется для элементов с минимальным скруглением, например, для маленьких иконок или бейджей.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-corner-extra-small` | 4px | Скругление всех углов |
| `--mat-sys-corner-extra-small-top` | 4px 4px 0 0 | Скругление только верхних углов |

## Full — Полное скругление

Используется для создания идеально круглых элементов, например, для аватаров, иконок в круге, FAB кнопок.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-corner-full` | 9999px | Полное скругление (круг) |

## Large — Большое скругление

Используется для элементов с заметным скруглением, например, для карточек, панелей, диалогов.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-corner-large` | 16px | Скругление всех углов |
| `--mat-sys-corner-large-end` | 0 16px 16px 0 | Скругление правых углов |
| `--mat-sys-corner-large-start` | 16px 0 0 16px | Скругление левых углов |
| `--mat-sys-corner-large-top` | 16px 16px 0 0 | Скругление верхних углов |

## Medium — Среднее скругление

Используется для элементов с умеренным скруглением, например, для кнопок, полей ввода, карточек.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-corner-medium` | 12px | Скругление всех углов |

## None — Без скругления

Используется для элементов с прямыми углами, например, для таблиц, разделителей, некоторых карточек.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-corner-none` | 0 | Без скругления |

## Small — Маленькое скругление

Используется для элементов с лёгким скруглением, например, для маленьких карточек, чипов, тегов.

| Переменная | Значение | Назначение |
|------------|----------|------------|
| `--mat-sys-corner-small` | 8px | Скругление всех углов |

## Рекомендации по использованию

1. **Full** — используйте для аватаров, иконок в круге, FAB кнопок
2. **Extra Large** — используйте для модальных окон, больших карточек
3. **Large** — используйте для карточек, панелей, диалогов
4. **Medium** — используйте для кнопок, полей ввода
5. **Small** — используйте для чипов, тегов, маленьких карточек
6. **Extra Small** — используйте для бейджей, маленьких иконок
7. **None** — используйте для таблиц, разделителей, прямоугольных элементов
8. **Directional токены** (start, end, top) — используйте для одностороннего скругления (например, для вкладок, тостов)
